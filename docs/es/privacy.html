Política de Privacidad

Fecha de Vigencia: 22 de Diciembre, 2025

Ed-Toolkit valora tu privacidad. Esta política de privacidad explica cómo manejamos tus datos cuando usas nuestra aplicación de macOS.

1. Recolección de Datos

No recolectamos, almacenamos o transmitimos ningún dato personal o de uso. Todo el procesamiento se realiza localmente en tu dispositivo. La aplicación no se conecta a servidores externos, APIs o servicios de terceros.

2. Funcionalidad Sin Conexión

La aplicación funciona completamente sin conexión. Tu texto, datos JSON y URLs se procesan de forma segura en tu Mac y nunca se transmiten por internet.

3. Sin Análisis o Seguimiento

No usamos ninguna forma de análisis, seguimiento o publicidad. Tus interacciones con la aplicación permanecen completamente privadas.

4. Permisos

La aplicación no solicita acceso a tus archivos, red, contactos o cualquier dato sensible. Solo usa recursos del sistema necesarios para el renderizado y procesamiento local.

5. Seguridad

Dado que toda la funcionalidad se ejecuta localmente y la aplicación no se comunica con sistemas externos, no hay riesgo de exposición de datos a través de transmisión de red.

6. Cambios a Esta Política

Podemos actualizar esta política en el futuro para reflejar nuevas características o requisitos legales. Cualquier actualización se publicará dentro de la aplicación o en nuestro sitio web oficial.
