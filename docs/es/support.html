Preguntas Frecuentes

¿Cómo cambio entre idiomas?

Ed-Toolkit usa automáticamente el idioma de tu sistema. Cambia la configuración de idioma de macOS en Preferencias del Sistema > Idioma y Región para alternar entre inglés, español e hindi.

¿Cómo uso los atajos de teclado?

Cada herramienta tiene atajos específicos:
• Codificador de URL: ⌘E para codificar, ⌘D para decodificar
• Formateador JSON: ⌘K para limpiar, ⌘F para formatear
• Comparador de Diferencias: ⌘R para comparar, ⌘B para regresar

¿Es Ed-Toolkit accesible con lectores de pantalla?

¡Sí! Ed-Toolkit tiene soporte completo para VoiceOver con etiquetas descriptivas, sugerencias y gestión adecuada del foco. Todas las características son completamente accesibles mediante navegación por teclado.

¿Qué formatos de archivo son compatibles?

Ed-Toolkit funciona con texto plano para todas las herramientas. El Formateador JSON soporta formato JSON estándar, y el Comparador de Diferencias puede comparar cualquier contenido de texto.

Características de Accesibilidad

  • VoiceOver: Soporte completo para lectores de pantalla
  • Navegación por Teclado: Navegación con Tab y teclas de flecha
  • Gestión de Foco: Manejo inteligente del foco
  • Alto Contraste: Respeta la configuración de accesibilidad del sistema
  • Multiidioma: Localizado para inglés, español e hindi
