Frequently Asked Questions

How do I switch between languages?

Ed-Toolkit automatically uses your system language. Change your macOS language settings in System Preferences > Language & Region to switch between English, Spanish, and Hindi.

How do I use keyboard shortcuts?

Each tool has specific keyboard shortcuts:
• URL Encoder: ⌘E to encode, ⌘D to decode
• JSON Formatter: ⌘K to clear, ⌘F to format
• Diff Matcher: ⌘R to compare, ⌘B to go back

Is Ed-Toolkit accessible with screen readers?

Yes! Ed-Toolkit has comprehensive VoiceOver support with descriptive labels, hints, and proper focus management. All features are fully accessible via keyboard navigation.

What file formats are supported?

Ed-Toolkit works with plain text for all tools. The JSON Formatter supports standard JSON format, and the Diff Matcher can compare any text content.
